## 📝 Reflection

My first year laid the essential technical and theoretical groundwork for my trajectory in data engineering. Mastering C++ syntax and algorithmic logic provided the disciplined problem-solving mindset necessary for structuring efficient SQL and NoSQL databases. Furthermore, diving into computer architecture and system design bridged the gap between raw hardware components and high-level software. This low-level understanding makes it significantly easier to conceptualize the scalable cloud infrastructure—such as AWS, Azure, and Google Cloud—that underpins modern data pipelines. 

Balancing these highly technical modules with UI/UX principles in Human-Computer Interaction also highlighted the importance of user-centric design. It reinforced the idea that raw data processing is only half the battle; presenting complex information clearly is equally vital, an approach I now carry into designing intuitive Power BI dashboards for end-users. Overall, this foundational year transformed abstract computing concepts into the practical building blocks I use for complex system development today.

— *Jing Yi*
