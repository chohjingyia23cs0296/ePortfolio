# ⚙️ Year 1 — Semester 1

## 📚 Courses
- [Digital Logic](./digital_logic)  
  Fundamentals of digital circuits, combinational and sequential logic, and practical lab exercises.

- [Programming Techniques I](./programming_techniques_I)  
  Introductory programming concepts, problem solving and basic software development practices and labs.

- [Technology and Information System](./technology_and_information_system)  
  Overview of information systems, IT fundamentals and tools used in system development and operations.

- [Integrity and Anti-Corruption Course](./integrity_and_anti_corruption_course)  
  Ethics, integrity and professional conduct in computing and engineering contexts.

## Semester Learning
In Semester 1 I established foundational skills:
- Core programming fundamentals: variables, control flow, functions and simple data structures through Programming Techniques I.
- Understanding hardware basics with Digital Logic: gates, flip-flops and combinational/sequential design.
- Overview of information systems and essential IT tools to support later coursework.
- Professional and ethical responsibilities via the Integrity and Anti-Corruption course.

## Reflection
Semester 1 provided the base for later coursework: programming habits, problem decomposition and engineering ethics. Key learnings included:
- Write small, testable programs and incrementally improve them.
- Map theoretical circuit concepts to hands-on lab work to deepen understanding.
- Document work and keep reproducible steps — it massively helps debugging and collaboration.
- Ethics are integral to engineering decisions, especially when handling data and systems.
