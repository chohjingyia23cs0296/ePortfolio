# ⚙️ Year 2 — Academic Portfolio

## 📚 Academic Overview
Year 2 builds core engineering foundations across algorithms, data, systems and software development. Coursework balances theory (data structures & algorithms, operating systems), practical systems work (database programming, network communication, system development), and data-focused classes (data analytics, data mining). Deliverables across the year include lab reports, projects, notebooks, and written assignments that demonstrate design, implementation, evaluation and documentation skills.

## 📝 Reflection
Year 2 was about turning core theory into reliable, testable practice. Key reflections:
- Reproducibility is essential: notebooks are great for exploration, but scripts, dependency lists and clear README files make results repeatable and sharable.
- Abstractions reduce complexity: OOP, good schema design, and modular components prevented duplication and simplified testing across projects.
- Measure and validate: debugging and profiling (for example in OS and data processing tasks) guided effective optimization decisions.
- Integration matters: combining knowledge from networking, databases, and software engineering yields more robust systems—understanding lower-level system behaviour leads to better higher-level designs.
