# 🌿 Year 2 — Semester 1

This semester marked a significant transition from learning isolated programming concepts to engineering interconnected, full-stack systems. The focus shifted heavily toward the practical application of software lifecycles, backend logic, data management, and the underlying network infrastructure required to support robust applications. 

## 📚 Courses

* [Data Structure and Algorithm](./Data%20Structure%20and%20Algorithm)
* [Database (WBL)](./Database)
* [Network Communication](./Network%20Communication)
* [Software Engineering](./Software%20Engineering)
* [System Development Technology](./System%20Development%20Technology)

## Semester Learning

Throughout this semester, I developed a comprehensive understanding of how data flows through modern software systems:
* **Algorithmic Efficiency:** In *Data Structure and Algorithm*, I learned to move beyond writing functional code to writing highly optimized code, mastering Big-O notation, dynamic memory, and complex structures like binary search trees and graphs.
* **Data Storage & Retrieval:** Through the *Database (WBL)* module, I gained hands-on experience designing relational database schemas, executing complex SQL queries, and understanding the normalization processes essential for maintaining data integrity.
* **Infrastructure & Connectivity:** *Network Communication* provided the foundational knowledge of how applications securely transmit data. I practiced configuring routers, subnetting IP addresses, and analyzing packet traffic to understand the TCP/IP and OSI models in practical scenarios.
* **System Lifecycle & Architecture:** In *Software Engineering*, I explored the formal methodologies (like Agile and Scrum) required to gather business requirements, map out UML diagrams, and plan a system's architecture before writing a single line of code.
* **Practical Implementation:** Finally, *System Development Technology* brought all these concepts together. I collaborated in a team environment to build a functional, full-stack web application, connecting frontend interfaces to backend servers and live databases using version control (GitHub).

## Reflection

Looking back, this semester was incredibly foundational for my path in Data Engineering. Previously, I viewed programming, databases, and networking as separate domains. Now, I see them as an integrated ecosystem. 

Building a complete software application from scratch taught me that data doesn't just exist; it has to be efficiently collected via frontend forms (*System Development*), transmitted securely across subnets (*Network Communication*), structured logically in memory (*Data Structures*), and persisted reliably in relational tables (*Database*). Understanding this entire lifecycle makes it significantly easier to architect scalable cloud data pipelines and design enterprise-level dashboards today. It shifted my mindset from being just a programmer to becoming a systems thinker.

— *Jing Yi*
