# Data Analytic Programming

## Overview
The **Data Analytic Programming** module introduces the core principles and practical techniques used to clean, process, analyze, and interpret data through programming. Throughout this module, I worked with datasets, applied analytical methods, and used code to extract meaningful insights from raw information. As a second-year Data Engineering student at Universiti Teknologi Malaysia (UTM), this module strengthened my ability to think critically about data and gave me hands-on experience in applying programming to real analytical problems.

## Objectives
- To understand the workflow of data analysis from raw data to insight generation.
- To develop skills in data cleaning, preprocessing, and handling missing or inconsistent values.
- To perform exploratory data analysis (EDA) using descriptive statistics and visualizations.
- To apply programming tools such as Python to solve data-related problems in a structured way.
