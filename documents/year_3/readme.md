## 📝 Reflection
Year 3 emphasised integration: combining algorithms, systems knowledge and data engineering into reproducible projects. Major reflections:
- Reproducibility matters: notebooks + dependency manifests + clear README files make experiments re-runnable and trustworthy.
- Measure before optimizing: profiling (time/memory/IO) guided effective design decisions in data processing and crawling tasks.
- Systems-awareness improves application and pipeline design: understanding resource limits (OS, memory, concurrency) led to more robust solutions.
- Communication and documentation are as important as implementation — clear reports, diagrams and datasets make technical work accessible and defensible.
