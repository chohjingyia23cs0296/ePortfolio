## Reflection
This course was valuable because it shifted my perspective from writing isolated code to thinking in terms of **systems, stakeholders, and sustainability**.  
At first, I tended to focus more on producing outputs quickly, but over time I realized that enterprise development requires deeper planning, clear modelling, and alignment with real user/business needs.

One of my key personal takeaways is that **good system design is a communication tool**, not just a technical artifact.  
When models and documentation are clear, team members can collaborate better and implementation becomes more predictable.

The industry talks were especially meaningful because they showed me that what we learn in class is directly relevant to professional software engineering.  
Hearing practical experiences made me reflect on my own readiness and encouraged me to improve not only technical skills but also professionalism, adaptability, and continuous learning habits.

Overall, this subject helped me mature as a software engineering student by strengthening both my technical foundation and my mindset toward real-world enterprise development.

## Possible Improvements
Although I made meaningful progress, there are several areas I can improve further:

- **Deeper modelling practice**  
  I want to practice more advanced and detailed modelling techniques so I can represent complex systems more accurately.

  I can improve in extracting hidden constraints and edge cases earlier in the design phase.

- **More comprehensive project documentation**  
  Future submissions can include clearer rationale for each design decision and better traceability between requirements, models, and implementation.

- **More proactive application of industry insights**  
  I plan to connect lessons from talks directly into project strategy (e.g., maintainability and testing considerations from the beginning).

- **Continuous reflection during the semester**  
  Instead of reflecting only at the end, I should document reflections regularly to monitor growth and adjust learning strategies sooner.
