# Business Intelligence Project

## Overview
This repository contains the Business Intelligence (BI) project work completed across four phases: Phase 1, Phase 2, Phase 3, and the Final Phase. Each phase includes a PDF report documenting objectives, methodology, analysis, and outcomes. Phase 2 includes an Alteryx workflow and Phase 3 includes sample datasets used for analysis.

## Objectives
- Apply business intelligence techniques to real-world sample data to generate actionable insights.
- Practice end-to-end BI processes: project scoping, data collection, cleaning, transformation, analysis, and reporting.
- Demonstrate competence with data preparation tools (Alteryx workflow included) and spreadsheet-based data.
- Produce clear documentation and evidence (reports, workflow, datasets) supporting the analysis and recommendations.

## What I Did
- Scoped the BI project and defined key questions to investigate in Phase 1.
- Collected and prepared data, performing cleaning and transformation steps documented in Phase 2.
- Built and executed an Alteryx workflow to automate ETL tasks and ensure reproducible data preparation.
- Performed exploratory and descriptive analysis in Phase 3, generating summary tables and visualizations.
- Synthesised findings and drafted recommendations in the Final Phase report.

## Key Learning Outcomes
- Gained practical experience with the BI lifecycle: scoping, ETL, analysis, and reporting.
- Learned to use Alteryx for repeatable ETL processes and to prepare datasets for analysis.
- Improved skills in interpreting sales, orders, and payments data to derive business insights.
- Developed documentation habits to make analysis reproducible and findings accessible.

## Reflection
Working iteratively across phases helped refine the research questions and improve data quality before drawing conclusions. Using an Alteryx workflow accelerated the ETL process and reduced manual errors. The project highlighted the importance of clear documentation and a well-defined data dictionary for sharing results.

## Possible Improvements
- Add a data dictionary describing each dataset's columns and sample values to make datasets easier to use.
- Include exported visualizations (PNG/SVG) from Phase 3 so readers can quickly see key findings without opening PDFs.
- Provide alternative scripts (Python/SQL/R) that reproduce the Alteryx workflow for users without access to Alteryx.
- Link directly to important sections (page numbers or headings) inside the PDF reports for faster navigation.
